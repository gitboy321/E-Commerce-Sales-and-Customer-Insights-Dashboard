E-Commerce Sales and Customer Insights Dashboard
Project Overview

This project focuses on analyzing e-commerce order data to derive meaningful insights related to sales performance, customer behavior, and business trends. The dataset was cleaned, transformed, and analyzed using Excel, SQL, Python, and Power BI.

Objectives

Clean and preprocess raw data
Identify and handle missing values and duplicates
Perform data analysis using SQL and Python
Build an interactive Power BI dashboard
Generate insights on revenue, customers, and product categories

Tools & Technologies Used

Excel / Power Query – Data cleaning and preprocessing
SQL Server (SSMS) – Data querying and analysis
Python (Pandas & CSV module) – Data transformation and validation
Power BI – Data visualization and dashboard creation

Key Features

Data cleaning (missing values, duplicates, formatting)
Revenue calculation and analysis
Category-wise and city-wise performance analysis
Customer insights and top performers
Monthly sales trend analysis
Interactive dashboard with slicers and filters

Dashboard Highlights

Revenue by Category
Revenue by City
Monthly Sales Trends
Top Customers
Rating Distribution

Project Workflow

Data Cleaning using Excel & Power Query
Data Analysis using SQL
Data Processing using Python
Dashboard Creation using Power BI

Outcome

The project provides actionable insights into sales performance and customer trends, helping businesses make data-driven decisions.

Files Included

Cleaned dataset (Excel/CSV)
SQL scripts
Python scripts / Notebook
Power BI dashboard (.pbix)
Project report

Conclusion

This project demonstrates end-to-end data analysis workflow, from raw data cleaning to interactive dashboard creation, showcasing practical data analytics skills.
